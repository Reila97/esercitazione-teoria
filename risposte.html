Che cosa è un algoritmo?
un algoritmo è un insieme finito di passi che permettono di risolvere un problema, in poche parole, l'algoritmo dice alla macchina cosa e come farlo.
Sono il mattone dei programmi informatici.
Poiché il computer parla in codice binario, questi algoritmi, per essere "tradotti", necessitano dei linguaggi di programmazione.
( es. Javascript )
Cosa è una variabile?
le variabili sono dei "contenitori" che aiutano a gestire i dati durante la programmazione.
Tipo una scatola con un’etichetta sopra.
Sono totalmente riutilizzabili, hanno nomi unici e devono essere usati in punti precisi del codice (scope limitato).
Li si riconosce dalla loro dichiarazione "let" seguita dal nome.
( es. let nomeDellaVariabile = *** )
Undefine e null sono la stessa cosa?
quando si crea una variabie ci sono due possibilità: attribuirle subito un valore, oppure lasciarla vuota.
Se la vogliamo creare gia vuota, useremo il valore "undefined"
( es. let nomeDellaVariabile = undefined ),
se le vogliamo attribuire un valore subito, è comunque possibile svuotarla successivamente usando la parola "null"
( es. let nomeDellaVariabile = null )
